Running Scripts

Transmission can be set to invoke a script when downloads complete. The environment variables supported are:

  • TR_APP_VERSION
  • TR_TIME_LOCALTIME
  • TR_TORRENT_DIR
  • TR_TORRENT_HASH
  • TR_TORRENT_ID
  • TR_TORRENT_NAME

More information is available on the Transmission wiki.
